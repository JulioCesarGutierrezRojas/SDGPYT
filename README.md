# Proyecto Fullstack con Spring Boot y React

Este proyecto está basado en una arquitectura fullstack utilizando Spring Boot como backend y React como frontend.

## Tecnologías utilizadas

### Backend
- **Spring Boot**: Framework para construir aplicaciones web y APIs.
- **Java**: Versión 21.
- **JDK**: Versión 21.

### Frontend
- **React**: Biblioteca para construir interfaces de usuario.
- **Vite**: Herramienta para el desarrollo y bundling del frontend.
- **Node.js**: Versión 22 o superior.
- **npm**: Para gestionar las dependencias de Node.js.

## Requisitos previos

Asegúrate de tener instalado en tu entorno de desarrollo:

### Para el backend:
- JDK 21
- Maven

### Para el frontend:
- Node.js v22 o superior (descargar de [Node.js official website](https://nodejs.org/)).
- npm (incluido con Node.js).
